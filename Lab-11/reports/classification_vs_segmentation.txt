CLASSIFICATION VS SEMANTIC SEGMENTATION
=======================================

1. What does classification output?
One label for the whole image (for example: "forest").

2. What does segmentation output?
A class label for every single pixel in the image.

3. Which task is more suitable for land-cover mapping?
Segmentation. We need to know exactly where the forest or river is, not just that it exists.

4. Which task is more suitable for quick scene categorization?
Classification. It is fast and simple.

5. Why is segmentation more computationally demanding?
The model must predict thousands of labels (one for each pixel) instead of just one word.

6. How could segmentation support mission operations?
It can precisely map flooded areas or find safe landing zones for drones.
